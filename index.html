My journey into Tech


Roughly four years ago during my undergraduate in Economics, I first touched code when I decided it would be useful to learn Java as a universal programming language. Needlees to say, I've been hooked ever since. Following that, I have self-taught myself (responsive) web development, Python, more Java and a lot of other skills like wireframing and prototyping.

I was lucky enough to also get the opportunity to complete a MSc in Computer Science at the University of Edinburgh which allowed me to learn about Human-Computer Interaction, Computer Networking, Security and of course Software Engineering, Modelling, Software Architecture.

Technology and Programming are at the fore-front of everything I do and I also volunteer to teach programming to others as a Web Development Instructor at CodeFirst: Girls and co-lead a weekly freeCodeCamp meetup in Edinburgh to share my knowledge and enthusiasm.
