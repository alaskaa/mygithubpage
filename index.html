Simon aims to test its player on remembering a sequence. The game can be played in either normal mode or strict mode. Each time the player correctly plays back a sequence after listening, the count goes up and the level of difficulty increases. The game is written in HTML, CSS, CSS Grid and JavaScript and makes use of async/await to deal with user input.

Tic-Tac-Toe is a well known 2 player game that a human player can play against the computer. This version of the game is unbeatable since the computer is using the minimax algorithm to predict the human players scores and compares each game state against each other. Try and win if you like (it's impossible)! Tic-Tac-Toe is written in HTML, CSS and JavaScript.

Battleships is a terminal application that Marcus Lancaster and I built entirely in Python. It is based on the famous game Battleships but played entirely on the terminal. My journey into Tech


Roughly four years ago during my undergraduate in Economics, I first touched code when I decided it would be useful to learn Java for Android Development. Following that, I have self-taught myself basic web development skills and some JavaScript. Through my determination and continuous efforts in programming I was accepted into the Computer Science Masters Programme at the University of Edinburgh, which gave me the opportunity to increase my programming abilities and learn about Software Design, Architecture, Management and also Computer Networking and Security. I also taught myself Python during the course of this degree which I now also use for data analysis.
